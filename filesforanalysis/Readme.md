# filesforanalysis

PDFs of **dependency traces** generated from the Tamarin prover's interactive mode.

**Generate:** Run `make web` or `make web_proofs`, open a proof in the GUI, and export the dependency graph as PDF.

**Docker:** `make docker-web` or `make docker-web-proofs` — access at http://localhost:3001 (or `PORT=8080` if needed).
